# Signal Leakage Attack 

This software implements a signal leakage attack against the protocols DXL [DXL12].


Run attacks against DXL [DXL12]:
- `make`
- `python3 NewAttack.py <\lambda>`

For example, python3 NewAttack.py 4

- for comparing with the attack [BSV21], run `python3 Bindel_Attack.py`
- for comparing with the attack [QDC+22], run `python3 Qin_Attack.py`
